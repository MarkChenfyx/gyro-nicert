两仪四象量化策略
两仪四象量化策略是一套 “市场效率 → 动态周期 → 量价动能”​ 的自适应趋势跟踪系统：
趋势越强，周期越短、反应越快；震荡越久，周期越长、过滤噪音
最终用 成交量加权价格动能（两仪四象）​ 决定多空
核心逻辑
EF —— 市场效率系数（Market Efficiency）
EF := MAX(HHV(CLOSE,N1) - LLV(CLOSE,N1),ABS(CLOSE - REF(CLOSE,N1)))
/ SUM(ABS(CLOSE - REF(CLOSE,1)), N1);
它在算什么？
分子：
要么是 N1 周期内的最大价格跨度
要么是当前价格相对 N1 前的绝对变化
取两者中更大的一个
分母：
N1 天内所有“每日波动绝对值”的总和
相当于“走的总路程”
直观理解
EF值	市场状态
≈ 1	价格几乎直线走，强趋势
≈ 0.5	趋势 vs 震荡临界点
≈ 0	原地打转，完全无趋势
EF 是对“趋势纯度”的度量
EFMA —— 平滑后的市场效率
EFMA := EMA(EF, N1);
对 EF 做指数平滑
防止短期噪声导致频繁切换
更像“当前市场的结构性效率”
MER —— 动态自适应周期（最关键）
MER := INTPART(N1 - (EFMA - 0.5) * N2);
逻辑含义
EFMA > 0.5（趋势强）
→ (EFMA - 0.5)为正
→ MER 变小​
更快响应趋势
EFMA < 0.5（震荡）
→ (EFMA - 0.5)为负
→ MER 变大​
拉长周期，减少假信号
一个非常经典的自适应周期设计
LY —— 成交量加权的价格动能
LY := SUM((CLOSE - REF(CLOSE,1)) * VOL, MER);
它在算什么？
每一天的涨跌 × 成交量
在 MER 动态周期​ 内累加
本质是：
“资金推动的真实方向”
LY > 0	上涨有量
LY < 0	下跌有量
LYSX —— 两仪四象（策略核心信号）
LYSX := EMA(LY, N1);
对 LY 再做一次平滑
得到最终多空方向指标
LYSX	含义
> 0	多头占优
< 0	空头占优
交易规则
LYSX > 0, BPK;   // 平空 + 做多
LYSX < 0, SPK;   // 平多 + 做空
始终在场
无观望
