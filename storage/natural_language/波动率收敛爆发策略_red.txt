波动率收敛爆发策略
波动率收敛后常预示大行情
这叫 Volatility Contraction → Expansion（波动率收缩→扩张）
市场共识逻辑是：
价格进入窄幅震荡 / 筹码沉淀
多空力量暂时平衡
一旦被打破（突破支撑/阻力/通道），积压的未平衡订单集中释放
技术派常说的：
"The longer the coil, the harder the spring."
（横得越久，爆得越猛）
本策略是一个基于波动率收敛 + 自适应抛物线跟踪止损止盈（类 Parabolic SAR）的波段策略
用布林式通道判断趋势方向，低波动时进场，用加速递增的抛物线跟踪止损来止盈，跌破通道边界做硬止损。
核心逻辑
MIDLINE := MA(CLOSE, 20);
STD_    := STDP(CLOSE, 26);
UPBAND  := MIDLINE + 2 * STD_;
LOWBAND := MIDLINE - 2 * STD_;
通道历史极值
MAX_VAR := HHV(UPBAND, 26);   // 过去26期上轨最高值
MIN_VAR := LLV(LOWBAND, 26);  // 过去26期下轨最低值
不是价格的高低点，而是通道本身的极值
这相当于一个"扩张后的通道天花板/地板"，用来判断价格是否有效突破震荡区间
开仓条件
多头开仓
LLV(STD_, 7) < 18          //  波动率收缩
波动率压缩后，价格放量突破通道上轨 → 认为趋势启动，做多。
	
空头开仓
LLV(STD_, 7) < 18
对称逻辑，向下突破做空。
	
硬止损
如果价格反向击穿通道极值，说明突破失败/趋势反转，无条件离场。
跟踪止盈
这部分是策略最复杂也最有意思的地方——类似 Parabolic SAR 但做了自定义改造
多头跟踪止损逻辑
PARCLOSE 会逐步向最高价靠拢，价格涨得越高，跟得越快
价格回踩到跟踪线下方 → 止盈离场
